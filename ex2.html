4.   Regex Engine

Implementaion of a Nondeterninistic finite automata(NFA) and Deterministic Finite Automata (DFA) to create a regular expression engine. The implementation folows the formal defination of NFA and converts the NFA to DFA to accept or decline inputs that doesn't meet the specifid regular expression. The functional programing language called OCAML is used for the implementaion.

5.   Small C Interpreter

Inplementation of a C compiler using the functional programing OCAML. This project uses methods such as parsing and lexing to tokenize the C language inputs and turn them into tokens of OCAML familiar languages. Then excutes the tokens. The program does mathimatical computations and basic control structures.
